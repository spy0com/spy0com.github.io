2018-12-28

##Centos 6.x 防火墙操作

关闭命令： service iptables stop

永久关闭防火墙：chkconfig iptables off

两个命令同时运行，运行完成后查看防火墙关闭状态
service iptables status

开放端口：
/sbin/iptables -I INPUT -p tcp --dport 80 -j ACCEPT &#x5199;&#x5165;&#x4FEE;&#x6539;

/etc/init.d/iptables save &#x4FDD;&#x5B58;&#x4FEE;&#x6539;

service iptables restart &#x91CD;&#x542F;&#x9632;&#x706B;&#x5899;&#xFF0C;&#x4FEE;&#x6539;&#x751F;&#x6548;

关闭端口：
/sbin/iptables -I INPUT -p tcp --dport 80 -j DROP &#x5199;&#x5165;&#x4FEE;&#x6539;

/etc/init.d/iptables save &#x4FDD;&#x5B58;&#x4FEE;&#x6539;

service iptables restart &#x91CD;&#x542F;&#x9632;&#x706B;&#x5899;&#xFF0C;&#x4FEE;&#x6539;&#x751F;&#x6548;
